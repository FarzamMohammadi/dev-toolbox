# Entry Template

Use this structure for each diary entry. All sections are optional except Date and Title.

---

## [YYYY-MM-DD] - [Brief Title]

### Intention
What I set out to accomplish and why.

### Work Done
Concise summary of completed work.

### Decisions & Rationale
Key choices made and the reasoning behind them.

### Tools & Techniques
Technologies, approaches, patterns, or methods used.

### Learnings
New insights, skills developed, patterns discovered.

### Reflections
Challenges faced, open questions, or next steps.

## Usage Notes

### Customization Points

- **Sub-questions**: Number varies based on query complexity (3-7)
- **Source cards**: Include detailed cards for Tier 1-2 and controversial sources
- **Confidence map**: Include all claims rated, not just key findings
- **Counter-evidence**: Never leave empty

### Depth-Specific Adjustments

| Depth | Executive Summary | Full Report | Source Cards |
|-------|-------------------|-------------|--------------|
| QUICK | 1 paragraph | Brief per question | Optional |
| STANDARD | 1/2 page | Full per question | Key sources only |
| DEEP | 1 page | Comprehensive | All Tier 1-3 |
| EXHAUSTIVE | 1 page | Exhaustive | All sources |
